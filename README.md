## 📦 Descripción

**easyStockSuite** es una aplicación fullstack de gestión comercial que permite administrar inventario, ventas, compras, proveedores y clientes desde una interfaz web moderna.

Backend en **Java con Spring Boot** expuesto como API REST, frontend en **Angular 17**, base de datos **PostgreSQL** y despliegue completo con **Docker Compose**.

---

## 🛠️ Tech Stack

| Capa | Tecnología |
|------|-----------|
| Backend | Java · Spring Boot · Spring Security · JWT |
| Frontend | Angular 17 · TypeScript · HTML/CSS |
| Base de datos | PostgreSQL · Flyway (migraciones) |
| Infraestructura | Docker · Docker Compose |
| Arquitectura | REST API · Clean Architecture |

---

## ✨ Funcionalidades

- ✅ Gestión de productos e inventario (stock, alertas)
- ✅ Módulo de ventas y compras
- ✅ Gestión de proveedores y clientes
- ✅ Autenticación segura con Spring Security + JWT
- ✅ API REST documentada
- ✅ Frontend SPA con Angular 17
- ✅ Migraciones de base de datos con Flyway
- ✅ Despliegue con Docker Compose

---

## 🚀 Inicio rápido

```bash
# Clonar el repositorio
git clone https://github.com/crishof/easyStockSuite.git
cd easyStockSuite

# Levantar con Docker Compose
docker-compose up -d
```

La aplicación estará disponible en `http://localhost:4200`  
La API REST en `http://localhost:8080/api`
